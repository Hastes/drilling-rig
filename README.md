# ПРОМБУРСЕРВИС — Лендинг

Лендинг буровой компании: геологические изыскания, гидрогеология, тампонаж.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Изображения

Placeholder SVG находятся в `public/assets/`. Замените на реальные фото:
- `parallax-1.svg` — Hero
- `activities.svg` — Ключевые направления
- `map.svg` — География
- `objects.svg` — Объекты
- `projects.svg` — Проекты
- `equipment.svg` — Техника
- `documents.svg` — Документация
- `news.svg` — Новости

См. `public/assets/README.md` для деталей.

## Деплой на GitHub Pages

1. **Settings → Pages** → **Source** = **GitHub Actions**
2. Пуш в `main` запускает деплой. Сайт: https://пром54.рф/ (custom domain)
