# Изображения для лендинга

Добавьте следующие файлы в эту папку:

- `parallax-1.jpg` — фоновое изображение для главного экрана (Hero)
- `activities.jpg` — фото для блока «Ключевые направления деятельности»
- `map.jpg` — карта регионов для блока «География»
- `objects.jpg` — фото объектов
- `projects.jpg` — фото реализованных проектов
- `equipment.jpg` — фото техники
- `documents.jpg` — визуализация документов
- `news.jpg` — фото с площадок / новости

Поддерживаются форматы: jpg, jpeg, png, webp
